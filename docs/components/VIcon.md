# VIcon
Easy to use component for custom svg icons
## Dependencies:

- Bootstrap 4

## Usage

```html
  <v-icon>audio</v-icon>
```
The string "audio" between the compoent's tags is the name of the svg icon in the assets/icons folder

## Parameters

| Name            | Schema   | Required | Default |
| --------------- | -------- | -------- | ------- |
| `spin`          | boolean  | no       | false   |
