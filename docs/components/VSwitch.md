# DataTable
Simple switch

## Dependencies:
- Bootstrap 4

## Usage
```html
  <v-switch :value="vswitchVariable" :disabled="true" />
```

## Parameters
| Name      | Schema                                                    | Required |
|-----------|-----------------------------------------------------------|----------|
| `value`   | boolean                                                   |    yes   |
| `disabled`| boolean                                                   |    no    |

## Events
| Name      | Output                                                    |
|-----------|-----------------------------------------------------------|
| `@input`  | value: boolean                                            |
